# 金山打字通游戏复刻
一个基于 **Qt6 + C++** 实现的经典金山打字通游戏复刻项目，专注于练习打字速度与指法，同时锻炼前端界面、后端逻辑、资源管理的完整软件开发能力。

## 项目介绍
本项目完整复刻了金山打字通内置的多款经典打字游戏，通过趣味化的游戏形式帮助用户练习键盘指法、提升输入速度与准确率。
项目重点实现 **UI 组件与后端逻辑解耦协调**，并合理整合图片、音效、背景音乐等多媒体资源，还原经典游戏体验。

## 游戏列表
- **KApple**：接苹果打字游戏
- **KFrog**：青蛙过河打字游戏
- **KMole2**：打地鼠打字游戏
- **KPolice**：警察抓小偷打字游戏
- **KSPACE**：太空大战打字游戏

## 技术栈
- C++
- Qt6（界面、事件、多媒体、资源管理）
- 键盘事件监听与输入判定
- 游戏逻辑：字符生成、碰撞检测、计分系统
- 资源管理：图片、音效、背景音乐加载与控制

## 编译与运行
1. 使用 Qt6 直接打开项目 `CMakeLists.txt`
2. 配置编译器并构建项目
3. 运行主程序，选择游戏开始练习
4. 根据游戏提示输入对应字符，完成游戏目标
