# 项目开发进度与架构说明
本项目采用**迭代式开发 + 统一架构演进**思路，逐步完善功能、扩展模块、优化结构，整体处于持续迭代优化阶段。

## 已完成内容（稳定版）
- **KApple / KFrog / KMole2**
  三大核心游戏已**全部开发完成**，代码经过完整整理与优化。
- 采用**统一通用框架**支撑所有游戏，架构高度复用。
- 采用**信号自下而上传递、函数自上而下调用**的清晰数据流模式。
- 分层职责明确：
  - `UILayer`：管理常驻UI、按键显示
  - `GameMap`：管理游戏场景渲染与逻辑更新
  - `PopWidget`：管理弹窗、提示、过渡界面

已将所有外部配置统一整理，方便快速修改与扩展：
